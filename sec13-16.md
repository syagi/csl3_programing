# 15.2
```
(* 最大公約数を求める *)
(* gcd: int -> int -> int *)
let rec gcd m n =
 if n = 0 then m
 else gcd n (m mod n)
(*test*)
let test1 = gcd 3 0 = 3;;
let test2 = gcd 27 18 = 9;;
```

停止性の議論
n=0の時の停止性は自明.  
nが0でない場合、gcd n (m mod n)となる.  
ここで、再帰i回目の引数をm_i  n_i、 m_i mod n_i を p_iとすると、
p_i は 0 <= p_i < n_i となる。  
定義より、 m_i+1 = n_i, n_i+1 = p_i  
よって再帰を繰り返す度に第二引数は小さくなり、いずれ0 となり停止する。

    [] -> []
